# UzhavanGo KivyMD Frontend

Run from project root:

```bash

```

- Light minimal UI with generous spacing.
- Animated 5-star review interaction.
- Receipt screen style aligned to web branding.
- Notification bell placeholder for API integration (`/api/v1/notifications/*`).
